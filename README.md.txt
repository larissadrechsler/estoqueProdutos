Título do projeto: Estoque de Produtos
Status do projeto: em desenvolvimento
Tecnologias que serão aplicadas: Java, Spring WEB, Thymeleaf, JPA
Time de desenvolvedores: Larissa
Objetivo do software: Realizar o cadastro de produtos contendo nome da peça, tamanhos e quantidades disponíveis e o preço unitário. 
Fornecer uma tabela listando todos os produtos disponíveis em estoque.
Funcionalidades do sistema (requisitos):

Requisitos Funcionais:
RF01 - Cadastro de produtos
RF02 - Listagem de produtos
RF03 - Edição de produtos
RF04 - Exclusão de produtos

6. Requisitos Não Funcionais:
RNF01 - Interface intuitiva e amigável
RNF02 - Desempenho rápido na pesquisa e carregamento
RNF03 - Segurança dos dados do usuário
RFN04 - Compatibilidade com diferentes sistemas operacionais (Windows, macOS, Linux)
RFN05 – Acesso à internet
